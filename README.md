# Breathe ESG Dashboard

A Django + React based ESG data ingestion and analyst review dashboard.

## Features

* Upload and manage ESG company data
* Source selection (SAP, Utility, Travel)
* Analyst review dashboard
* Approve and delete records
* Search companies
* CSV upload support
* Data visualization charts
* ESG styled UI

## Tech Stack

### Frontend

* React
* Tailwind CSS
* Recharts

### Backend

* Django
* Django REST Framework

## Source Types

* SAP Procurement/Fuel Data
* Utility Electricity Data
* Corporate Travel Data

## Run Locally

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
python manage.py runserver
```
